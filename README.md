# Quote Compare

Compare contractor quotes across renovation trades, track selections, and calculate total costs.

**Live demo:** https://mrcwbr.github.io/quote-compare

## Features

- Organize quotes by trade (plumbing, electrical, roofing, etc.)
- Add multiple contractor quotes per trade with price, date, document link, and notes
- Highlights the cheapest option automatically
- Select a preferred quote per trade
- Fixed summary footer with grand total across all selections
- Share state via URL or export as JSON
- Bilingual (German / English)

## Tech stack

- React 19, TypeScript, Vite
- Tailwind CSS
- i18next

## Development

```bash
npm install
npm run dev
```
